	\subsection{SRF08 Adresse ändern}
		Die aktuelle Adresse eines Moduls ist durch den Blinkcode beim Einstecken erkennbar. So kann jeweils überprüft werden, ob eine Adressänderung erfolgreich war.
		\begin{itemize}
		% Punktstyles ändern
		\renewcommand{\labelitemii}{$\circ$}
		\renewcommand{\labelitemiii}{$\cdot$}
			\item Falsche Adresse wird nicht geschrieben
			\begin{itemize}
				\item Nicht im gültigen Bereich:
				\begin{itemize}
					\item 0x00\dotfill $\Box$$\Box$%$\CheckedBox$
					\item 0xDE\dotfill $\Box$$\Box$%$\CheckedBox$
				\end{itemize}
				\item Ungerade:
				\begin{itemize}
					\item 0xE5\dotfill $\Box$$\Box$%$\CheckedBox$
					\item 0xFB\dotfill $\Box$$\Box$%$\CheckedBox$
				\end{itemize}
			\end{itemize}
			\item Gültige Adresse wird geschrieben:
			\begin{itemize}
				\item 0xFE\dotfill $\Box$$\Box$%$\CheckedBox$
				\item 0xE0\dotfill $\Box$$\Box$%$\CheckedBox$
			\end{itemize}
			\item Neue Adresse kann danach verwendet werden:
			\begin{itemize}
				\item 0xFE\dotfill $\Box$$\Box$%$\CheckedBox$
				\item 0xE0\dotfill $\Box$$\Box$%$\CheckedBox$
			\end{itemize}
		\end{itemize}
